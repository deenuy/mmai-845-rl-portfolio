# Presentation Handoff

This folder is the final presentation support pack.

## Start Here

1. Open `PPT_TEMPLATE_9_SLIDES.md`
2. Open `SLIDE_DECISION_SCRIPTS.md`
3. Open `SLIDE_ASSET_INDEX.md`
4. Build the deck using the images in `assets/`
5. Use `data/` if you want to cite precise values

## Included Files

- `PPT_TEMPLATE_9_SLIDES.md`
  - final 9-slide structure
- `SLIDE_DECISION_SCRIPTS.md`
  - decision / why / benefit notes for each slide
- `SLIDE_ASSET_INDEX.md`
  - final mapping from slides to images
- `PRESENTATION_NOTES.md`
  - framing and speaking guidance
- `assets/`
  - final slide images
- `data/`
  - supporting JSON metrics and assumptions
