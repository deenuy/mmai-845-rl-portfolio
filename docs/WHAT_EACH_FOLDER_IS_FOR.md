# What Each Folder Is For

## `src/`

Main Python source code.

Most important files:

- `paper_trading.py`
- `profile_store.py`
- `portfolio_environment.py`
- `features.py`

## `scripts/`

Runnable entry points.

Most important files:

- `run_product_dashboard.py`
- `seed_example_profiles.py`
- `export_phase_4_paper_trading.py`

## `artifacts/`

Saved outputs and frozen files.

Includes:

- frozen PPO model files
- sample SQLite profile database
- example HTML exports

## `docs/`

Detailed technical and product documents.

## `phase_reports/`

The long-form reports for each project phase.

## `project_docs/`

Roadmap and design specifications.

## `tests/`

Unit tests.
